# Service Template README

This is a template repository for creating new microservices in our platform.

## Features

- Pre-configured CI/CD pipeline
- Docker support
- Standard project structure
- Common utilities and helpers

## Getting Started

1. Use this template to create a new repository
2. Clone the new repository
3. Update the service name and configuration
4. Start building your service!

## CI/CD

The template includes a GitHub Actions workflow that:
- Runs tests
- Builds Docker images
- Deploys to staging environment
